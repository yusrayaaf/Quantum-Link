## Quick Start (Local Dev)

```bash
# 1. Install all dependencies
npm run install:all

# 2. Configure server environment
cp server/.env.example server/.env
# Edit server/.env with your values

# 3. Start dev servers (runs both concurrently)
npm run dev
```

- Client: http://localhost:5173  
- Server: http://localhost:3001

---

## Deploy to IONOS VPS (from Termux)

```bash
# Install SSH in Termux
pkg install openssh

# Make scripts executable
chmod +x deploy/setup.sh deploy/update.sh

# Full deploy (first time)
./deploy/setup.sh --host YOUR_VPS_IP --domain yourdomain.com

# Push updates (subsequent deploys)
./deploy/update.sh
```

The deploy script:
1. Installs Node 20, Nginx, Certbot, PM2 on the VPS  
2. Uploads and builds the project  
3. Configures Nginx with SSL (Let's Encrypt)  
4. Starts the app with PM2 (auto-restart on crash/reboot)  
5. Sets up UFW firewall

## Environment Variables

See `server/.env.example` for all options.

Key vars:
- `JWT_SECRET` вЂ” long random string (auto-generated by deploy script)  
- `CREATOR_PASSWORD` вЂ” admin panel password  
- `CORS_ORIGINS` вЂ” your domain(s)  
- `PORT` вЂ” server port (default 3001)
